# ❓ FAQ и Решение проблем

## ❓ Часто задаваемые вопросы

### Q: Как получить BOT_TOKEN?
**A:** 
1. Откройте Telegram
2. Найдите бота [@BotFather](https://t.me/BotFather)
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте полученный token

### Q: Где взять token для GitHub?
**A:**
1. Перейдите в [GitHub Settings](https://github.com/settings/tokens)
2. Нажмите "Generate new token"
3. Выберите нужные права доступа
4. Скопируйте токен (он больше не будет виден!)

### Q: Как изменить меню ресторана?
**A:** Отредактируйте файл `config.py`, найдите словарь `MENU` и добавьте/измените блюда.

### Q: Какой Python нужен?
**A:** Python 3.8 или выше. Проверьте: `python --version`

### Q: Можно ли использовать этот бот для другого ресторана?
**A:** Да! Просто измените данные в `config.py`.

## 🐛 Решение распространённых проблем

### Ошибка: "ModuleNotFoundError: No module named 'telegram'"

**Решение:**
```powershell
pip install -r requirements.txt
```

### Ошибка: "Invalid token provided"

**Решение:**
1. Проверьте, что токен правильно скопирован в `.env`
2. Убедитесь, что файл `.env` находится в папке проекта
3. Перезапустите бот

### Ошибка: "python: command not found"

**Решение:**
Используйте `python3` вместо `python` или добавьте Python в PATH.

### Бот не отвечает

**Проверьте:**
1. Бот запущен? (должен быть активный процесс)
2. Правильный ли токен?
3. Интернет соединение работает?
4. GitHub не заблокирован в вашей стране?

### Кнопки не работают

**Решение:**
Убедитесь, что используете последнюю версию:
```powershell
pip install --upgrade python-telegram-bot
```
