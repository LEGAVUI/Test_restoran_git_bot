# ⚡ Быстрый старт

## 5 шагов для запуска бота за 5 минут

### Шаг 1️⃣: Скачайте проект

```powershell
# Скопируйте папку telegram_restaurant_bot на ваш компьютер
```

### Шаг 2️⃣: Создайте файл .env

```powershell
# Отредактируйте .env.example и сохраните как .env
# BOT_TOKEN=YOUR_TOKEN_HERE  (скопируйте токен от @BotFather)
```

### Шаг 3️⃣: Установите зависимости

```powershell
# Откройте PowerShell в папке проекта
pip install -r requirements.txt
```

### Шаг 4️⃣: Запустите бота

```powershell
python main.py
```

Вы должны увидеть сообщение: "Бот запущен!"

### Шаг 5️⃣: Протестируйте в Telegram

1. Откройте Telegram
2. Найдите вашего бота по имени (например, @my_restaurant_bot)
3. Отправьте `/start`
4. Нажимайте кнопки и проверяйте функционал!


## 🐛 Отладка

### Если бот не запускается:

1. **Проверьте Python:**
   ```powershell
   python --version  # должен быть 3.8+
   ```

2. **Проверьте токен:**
   ```powershell
   # Откройте .env и убедитесь, что BOT_TOKEN правильный
   ```

3. **Проверьте интернет:**
   ```powershell
   ping google.com
   ```

4. **Проверьте логи:**
   Смотрите вывод консоли при запуске - там будут ошибки

### Если нет ошибок, но бот не отвечает:

```powershell
# Перезапустите бот (Ctrl+C и снова python main.py)
# Убедитесь, что бот активирован в @BotFather
```

## 📱 Тестирование в Telegram

### Создание тестового бота:

1. Напишите @BotFather в Telegram
2. `/newbot` - создать нового бота
3. Дайте ему имя для тестирования
4. Скопируйте токен
5. Вставьте в `.env`

### Полезные команды в @BotFather:

- `/mybots` - список ваших ботов
- `/setusername` - изменить username
- `/setdescription` - описание бота
- `/setcommands` - список команд
